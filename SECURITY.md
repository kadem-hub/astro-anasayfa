# Security Policy

## Supported Versions
Bu depo eğitim amaçlı bir landing page projesidir. Güvenlik yamaları ihtiyaç halinde uygulanır.

## Reporting a Vulnerability
Lütfen güvenlik açıklarını **public issue** olarak paylaşmayın.

Bildirimi şu kanallardan yapın:
- E-posta: info@ahmatematik.com (varsa) / veya GitHub hesabımdaki iletişim e-postası
- Konu: `[SECURITY] astro-anasayfa - <kısa özet>`

Bildiriminizde şu bilgileri ekleyin:
- Etkilenen sayfa/endpoint
- Reprodüksiyon adımları
- Beklenen vs gözlenen davranış
- Varsa PoC / ekran görüntüsü

## Response Timeline
Bildiriminizi aldıktan sonra:
- 48–72 saat içinde alındı onayı
- Uygun ise önceliklendirme ve düzeltme planı paylaşımı
